ERP연계

Cyber Branch란?(표준ERP연계 서비스 제공)

국내 금융권 최초로 출시된 Cyber Branch는 기업 내부에 서버를 구축하여 내부 시스템(인사/영업 시스템, ERP 등)과 안전하고 빠른 표준ERP연계 서비스주1) 를 제공하며, 2022년 3월 자금사고 예방솔루션주2) 을 탑재하여 안심경영이 가능합니다.

  • 주1) 표준ERP연계 서비스란?

    DB, FTP, RFC, SAP 등 어떤 엔진을 사용하든 연계가능한 표준ERP연계 서비스 입니다. (단, 표준 연계 외 범위는 추가 개발하여 제공)

    • 표준연계범위
      1. 1) 각 종 DB엔진 : MS-SQL, Oracle, SAP HANA, DB2, Sybase, MySQL 등
      2. 2) FTP 엔진 : FTP, sFTP, FTPs
      3. 3) RFC 엔진 : RFC6.XX, RFC710~760
      4. 4) SAP 엔진 : Odata, XI, PI, PO
      5. 5) API 엔진 : API HUB (각 종 Cloud ERP 지원)
  • 주2) 자금사고예방솔루션이란?
    • 이상거래감지

      Cyber Branch에 설정한 이상거래감지 Rule에 따라 이상거래감지 즉시 관리자 모바일로 통지되는 서비스 (서비스 종류 : 고액출금거래, 지정시간외거래, 신규계좌이체거래, 계좌변경)

    • AI안심경영보고서

      Cyber Branch 실제 금융정보 기반의 안심자금보고서 를 CEO, CFO 및 관리자 모바일로 통지하는 서비스(서비스 종류 : 일일시재보고서, 자금지급보고서)
