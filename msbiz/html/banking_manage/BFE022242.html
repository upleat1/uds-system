OTP 카드 안내 및 사용방법

  1. OTP 카드 앞쪽에 있는 [PRESS] 버튼을 누르면 일회용 비밀번호 부분에 6자리 숫자가 표시됩니다.
  2. 전자금융 거래 시 일회용 비밀번호 입력 단계에서 현재 카드에 나타나는 6자리 비밀번호를 입력해주세요.(반드시 현재 OTP 카드에서 보이는 6자리 숫자 입력)
  3. 분실/도난 시 은행에 즉시 신고해야 하며 본인의 관리 소홀로 발생한 손해는 은행이 책임지지 않습니다.
  4. OTP 비밀번호는 절대 타인에게 유출되지 않도록 주의해주세요.
  5. 공공기관(경찰, 검찰, 금융감독원 등)을 사칭하여 OTP 비밀번호를 요구하는 경우 100% 금융사기입니다.
