인증서 ID연결이란?

기업뱅킹서비스를 이용하기 전에 발급받은 모바일인증서를 사용할 ID를 설정해 주세요.

이용안내

  • 고객님의 개인사업자 ID를 KB국민인증서에 연결할 수 있습니다.
  • 여러 개의 개인사업자 ID를 보유한 고객님은 연결할 ID를 설정하신 후 해당 ID로 거래할 수 있습니다.
  • 발급받은 모바일인증서의 간편비밀번호 인증 후 연결 할 수 있으며, 간편비밀번호를 모를 경우 인증서를 재발급 해주세요.
    KB국민인증서 재발급
  • KB국민인증서 재발급 시 기존 KB국민인증서는 폐기됩니다.

절차 안내

  1. 간편비밀번호 인증

    간편비밀번호 6자리를 인증합니다.

  2. ID 정보 확인

    연결할 ID를 선택하고 해당 ID에 등록된 계좌를 확인합니다.

  3. ID 변경 완료

    KB국민인증서로 다시 로그인하면 변경 ID로 로그인 됩니다.
