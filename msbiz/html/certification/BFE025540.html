인증서 이용안내

  • 안전한 금융거래를 위하여 KB스타기업뱅킹 서비스는 인증서를 기반으로 이용하실 수 있습니다.
  • 고객님의 PC 또는 스마트폰에 저장된 인증서를 스마트폰에 복사하거나, 스마트폰에서 PC로 인증서를 복사하실 수 있습니다.

인증서 가져오기

PC에서 스마트폰으로

  1. 인증서 복사 메뉴 선택

    PC에서 스마트폰으로 PC에서 국민은행 홈페이지(www.kbstar.com)에 접속하여 인증센터>KB스타기업뱅킹 인증서 복사 메뉴를 선택합니다.

  2. 인증서 이동

    PC화면의 [PC>스마트폰 인증서 복사하기]를 클릭 후 [인증서 이동]을 선택합니다.

  3. 승인번호 생성

    가지고올 인증서를 선택 후 암호를 입력하면 PC화면에 승인번호가 생성됩니다.

  4. 가져오기 메뉴 선택

    인증서를 가지고 올 휴대폰에서 KB스타기업뱅킹 인증센터>인증서가져오기>PC에서 가져오기 메뉴를 선택합니다.

  5. 승인번호 입력

    PC에서 생성된 승인번호를 스마트폰 화면에 입력 합니다.

  6. 인증서 복사 완료

    PC로 이동완료된 인증서를 확인합니다.

스마트폰에서 스마트폰으로

  1. 스마트폰에서 가져오기 메뉴 선택

    스마트폰에서 KB스타기업뱅킹에 접속하여 인증센터>인증서 가져오기>스마트폰에서 가져오기 메뉴를 선택합니다.

  2. 승인번호 입력

    스마트폰 내보내기에서 복사할 인증서를 선택 및 비밀번호를 입력 후 생성된 승인번호를 인증서를 가져올 스마트폰에 입력합니다.

  3. 인증서 복사 완료

    스마트폰에서 이동완료된 인증서를 확인합니다.

인증서 내보내기

스마트폰에서 PC으로

  1. 인증서 내보내기 메뉴 선택

    인증서를 내보낼 휴대폰에서 KB스타기업뱅킹에 접속하여 인증센터>인증서 보내기>PC로 인증서 내보내기 메뉴를 선택합니다.

  2. 승인번호 생성

    내보낼 인증서를 선택 후 암호를 입력하면 휴대폰에 승인번호가 생성됩니다.

  3. 인증서 복사 메뉴 선택

    인증서를 가지고올 PC에서 국민은행 홈페이지(www.kbstar.com)에 접속하여 인증센터>KB스타기업뱅킹 인증서 복사 메뉴를 선택합니다.

  4. 인증서 이동 선택

    PC화면의 [PC>스마트폰 인증서 가져오기]를 클릭 후 [인증서 이동]을 선택합니다.

  5. 승인번호 입력

    휴대폰에서 생성된 승인번호를 PC화면에 입력합니다.

  6. 인증서 이동 완료

    스마트폰으로 이동완료된 인증서를 확인합니다.

스마트폰에서 스마트폰으로

  1. 스마트폰 내보내기 메뉴 선택

    스마트폰에서 KB스타기업뱅킹에 접속하여 인증센터>인증서 내보내기>스마트폰으로 내보내기 메뉴를 선택합니다.

  2. 승인번호 생성

    스마트폰 내보내기에서 복사할 인증서를 선택 및 비밀번호를 입력하면 승인번호가 생성됩니다..

  3. 스마트폰 가져오기 메뉴 선택

    인증서를 보낼 스마트폰에서 KB스타기업뱅킹 인증센터>스마트폰에서 가져오기 메뉴를 선택합니다.

  4. 승인번호 입력

    스마트폰에서 생성된 승인번호를 인증서를 보낼 스마트폰에 입력 합니다.

  5. 인증서 이동 완료

    스마트폰으로 이동완료된 인증서를 확인합니다.
