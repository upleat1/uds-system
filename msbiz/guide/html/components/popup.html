팝업
Popup Sample

1.알림, 컨펌 팝업

  • 단순 메세지성 컨펌 및 알림 팝업 : 네이티브로 작업. (퍼블 작업 없음)
  • 안내성(디자인) 레이어 팝업 : 퍼블 작업
  • 단순 메세지성 컨펌 및 알림 팝업 : 네이티브로 작업. (퍼블 작업 없음)
  • 안내성(디자인) 레이어 팝업 : 퍼블 작업

2.바텀시트 정책

  • 일반 바텀시트 : max값 95%만 적용하며 높이는 컨텐츠길이만큼 자동으로 지원. 컨텐츠가 95%이상이면 스크롤 자동 지원
사용법
Bottm Sheet
기본
팝업 ID 기준으로 호출하기때문에 팝업끼리 ID가 중복되면 안됨. 중복일 경우 변경
팝업의 ID는 임시ID(개발에서 변경 사용 가능)
Popup Open
아래와 같이 함수 호출 
    UICommon.Layer.open('#팝업ID');
콜백이벤트 삽입 
    UICommon.Layer.open('#팝업ID', function(){
        //팝업 오픈후 콜백이벤트 삽입
    });
Popup Close
아래와 같이 함수 호출 
    UICommon.Layer.close('#팝업ID');
콜백이벤트 삽입 
    UICommon.Layer.close('#팝업ID', function(){
        //팝업 닫은 후 콜백이벤트 삽입
    });
Popup안에 있는 버튼에 Popup Close기능을 주려면 'ui_pop_close' 클래스 추가
'ui_pop_close' 클래스는 해당 용도가 아니면 에러를 발생할수 있으니 안쓸때는 삭제
딤 클릭 이벤트 방지
'bg_no_click' 클래스를 Popup 최상단에 엘리먼트에 추가
닫기버튼 삭제 
    UICommon.Layer.open('#팝업ID','','noClose');
타이틀 삭제 
    UICommon.Layer.open('#팝업ID','noTitle');
타이틀 변경 
    UICommon.Layer.open('#팝업ID','타이틀변경텍스트입력');
Alert
기본 Alert 호출 
    UICommon.Layer.alert('얼럿 메세지');
Confirm
기본 Confirm 호출 
    UICommon.Layer.confirm('Confirm 메세지');
